配置参数说明

表格的参数定义在 jQuery.fn.treeView.DEFAULTS。

# 配置项 类型 默认值 可选值 说明
data array [] array 树数据
enableLink boolean false
false: 否
true: 是
树节点是否允许超链接
enableTopSwitch boolean false
false: 否
true: 是
是否开启顶部切换
